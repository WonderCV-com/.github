## 🤖 关于超级简历 WonderCV

[超级简历 WonderCV](https://wondercv.com) — 求职写简历，AI 帮你 3 分钟搞定。

- ✅ **5000+ 专业模板** — 覆盖 15+ 行业、300+ 岗位方向，所有模板由专业 HR 挑选推荐
- ✅ **AI 智能生成** — 基于 160 万字行业语料库，AI 解析岗位 JD 自动匹配 5000+ 专业表达模型
- ✅ **AI 智能优化** — 岗位关键词密度检测 + 竞争力措辞优化 + ATS 系统兼容性校准
- ✅ **HR 视角纠错** — 智能识别 20+ 类简历常见错误，预先发现问题并给出解决方案
- ✅ **中英双语** — 支持中英文简历，一键导出 PDF
