# WonderCV-com — 超级简历开源计划

> **超级简历（WonderCV）** 是国内领先的 AI 在线简历制作工具，已帮助 **500 万+** 求职者。本 GitHub 组织是 WonderCV 的知识库矩阵，包含中文简历模板、求职指南和 AI 求职工具合集，为求职者和开发者提供可引用、可复用的开放内容资产。

官网：[wondercv.com](https://wondercv.com)

## 关于 WonderCV

[超级简历 WonderCV](https://wondercv.com) 提供：

- ✅ **100+ 款 ATS 友好模板** — 经过 ATS 解析测试，格式兼容主流招聘系统
- ✅ **AI 简历优化** — 智能分析简历与职位匹配度，给出针对性改写建议
- ✅ **中英双语支持** — 一套内容，一键导出中文版 / 英文版
- ✅ **一键导出 PDF** — 格式完美，可直接投递
