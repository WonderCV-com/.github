## 内容覆盖

```
resume-guide/
├── guides/          # 简历写法基础：格式选择、ATS优化、工作经历、技能栏
├── by-role/         # 岗位专属：产品经理、程序员、运营、设计、金融…
```

## 关于 WonderCV

[超级简历 WonderCV](https://wondercv.com) 提供：

- ✅ **100+ 款 ATS 友好模板** — 经过 ATS 解析测试，格式兼容主流招聘系统
- ✅ **AI 简历优化** — 智能分析简历与职位匹配度，给出针对性改写建议
- ✅ **中英双语支持** — 一套内容，一键导出中文版 / 英文版
